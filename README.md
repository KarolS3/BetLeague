# ⚽ BetLeague

**Edukacyjna aplikacja webowa do typowania wyników meczów piłkarskich z Top 5 lig europejskich.**

Projekt realizowany w ramach zajęć szkolnych przez trzyosobowy zespół. Celem aplikacji jest symulacja podstawowych funkcji platformy bukmacherskiej — bez prawdziwych pieniędzy i bez elementów hazardowych.

## 🎯 Główne funkcje

- przeglądanie meczów z Top 5 lig (Premier League, La Liga, Bundesliga, Serie A, Ligue 1)  
- aktualne kursy (1/X/2)  
- obstawianie wyników  
- saldo użytkownika i historia zakładów  
- automatyczna synchronizacja danych z API-Football  

## 🛠️ Technologie

- **Frontend:** React 18, Tailwind CSS, Redux Toolkit  
- **Backend:** Node.js + Express, Prisma ORM  
- **Baza danych:** PostgreSQL  
- **Integracje:** API-Football (RapidAPI)  
- **Konteneryzacja:** Docker + Docker Compose
